ក្រុមការងារអង្គការ Sisters of Code បន្តការបណ្ដុះបណ្ដាលដើម្បីលើកកម្ពស់សមភាពយេនឌ័រនៅក្នុងប្រទេសកម្ពុជា

ខេត្តតាកែវ៖ នារសៀលថ្ងៃទី១១ ខែតុលា ឆ្នាំ២០២៥ ចាប់​ពីម៉ោង ២៖០០ ដល់ ៤៖៣០នាទីរសៀល ក្រុមការងារ Sisters of Code ចំនួន ៣​រូប បានចុះធ្វើការបណ្ដុះបណ្ដាលលើមេរៀនទី៩ ស្ដីពី Wix Website នៅមជ្ឈមណ្ឌលបច្ចេកវិទ្យាសហគមន៍ចំនួន ៣ទីតាំង រួមមាន វិទ្យាល័យ សុខអាន ត្រាំខ្នារ វិទ្យាល័យ ប៊ុនរ៉ានី ហ៊ុន សែន បាទី​ និង វិទ្យាល័យ ហ៊ុន សែន ១​ មិថុនា ដែលមានសិស្សានុសិស្សចូលរួមសរុបចំនួន ៣៥រូប។

កម្មវិធីបណ្តុះបណ្តាលនេះ​បានរៀបចំឡេីងក្នុងគោលដៅផ្តល់ការគាំទ្រអាទិភាពដល់​សិស្សស្រី ដើម្បីពង្រឹងសមភាពយេនឌ័រតាមរយៈការកសាងសមត្ថភាពបច្ចេកវិទ្យាលឌីជីថល និងជាការរួមចំណែកជំរុញអក្ខរកម្មឌីជីថលនៅក្នុងប្រទេសកម្ពុជា សំដៅពន្លឿនការធ្វើបរិវត្តកម្មឌីជីថលលើជំនាញបច្ចេកវិទ្យា ប្រកបដោយបរិយាបន្ន ចីរភាព និងប្រសិទ្ធភាព។

សូមជម្រាបជូនថា ម.ប.ស. នៅតាមវិទ្យាល័យសាធារណៈ គឺជាចក្ខុវិស័យវែងឆ្ងាយរបស់ ក.ប.ទ. លើការអភិវឌ្ឍវិស័យឌីជីថល ដើម្បីបម្រើដល់ការអប់រំទំនើប និងអភិវឌ្ឍមូលធនមនុស្សឌីជីថលនៅក្នុងមូលដ្ឋាន ទាំងក្នុងវិស័យសាធារណៈ និងឯកជន ដើម្បីរៀបចំប្រព័ន្ធអេកូឡូស៊ីឌីជីថលមួយដែលរឹងមាំ និងមានភាពរីកចម្រើននៅក្នុងសហគមន៍ទាំងមូល។ 

សម្រាប់ព័ត៌មានបន្ថែម សូមចូលទៅកាន់គេហទំព័រផ្លូវការរបស់ ម.ប.ស. គឺ www.ctc.edu.kh ដើម្បីតាមដានការវិវឌ្ឍន៍ថ្មីៗ។
